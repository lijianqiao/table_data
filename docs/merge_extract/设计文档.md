# 合并处理应用逻辑

## 使用方式

### 基本流程
1. **上传文件**: 选择多个CSV/Excel文件
2. **配置预处理** (可选):
   - 启用数据预处理
   - 选择清理选项
   - 选择去重选项
3. **开始处理**: 点击"开始合并处理"按钮
4. **选择去重列** (如果启用去重): 选择用于判断重复的列
5. **选择导出列**: 选择需要的数据列
6. **导出文件**: 选择格式并下载

### 预处理功能
- **数据清理**: 自动清理空值、空格等
- **数据去重**: 移除重复数据行
- **列级去重**: 基于指定列判断重复

### 导出功能
- **多格式支持**: XLSX/XLS
- **自定义命名**: 指定导出文件名
- **即时下载**: 生成后立即可下载
